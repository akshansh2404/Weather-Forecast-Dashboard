🌦️ Weather App

A modern and animated weather application built using HTML, CSS, and JavaScript that fetches real-time weather data using the OpenWeatherMap API.
The app features dynamic backgrounds, animated UI elements, glassmorphism effects, floating clouds, and responsive weather cards.

✨ Features

🔍 Search weather by city name

🌡️ Real-time temperature display

💧 Humidity information

🤗 "Feels Like" temperature

☁️ Dynamic weather emojis

🎨 Dynamic background changes based on weather conditions

✨ Glassmorphism UI design

☁️ Floating cloud animations

🌌 Animated stars effect

❌ Error handling for invalid city names


🛠️ Technologies Used

HTML5

CSS3

JavaScript (Vanilla JS)

OpenWeatherMap API
